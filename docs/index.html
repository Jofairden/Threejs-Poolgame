Rotate the cue with keys A or D. Cue rotates faster if you hold.

Shoot by pressing spacebar. Hold spacebar to shoot with more power.

Player decides what type of ball to go for. If you score a striped bar initially, you will need to always score that type.

Player wins if all 7 balls are scored and then scores the black 8 ball.

Toggle debug with F1. Go back to menu with ESC
